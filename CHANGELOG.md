# Changelog

All notable changes to this project will be documented here.

The format is based on Keep a Changelog, and the project uses semantic versioning.

## [1.0.0] - 2026-06-20

### Added

- Public Google Apps Script booking form
- Google Sheets booking storage
- Google Calendar availability checks and event creation
- Configurable appointment and buffer durations
- Standard provisional booking workflow
- Manual booking confirmation menu action
- Customer and owner email notifications
- Optional Stripe Checkout payment workflow
- Automatic Stripe payment reconciliation trigger
- Setup sidebar and iframe generator
- New-user documentation and troubleshooting guide
- GitHub repository documentation and basic secret scanning workflow
